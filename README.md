----- Business Financial Status Tracker -----

This is a beginner Python project that helps businesses track their assets, revenue streams, and expenses. The program calculates total amounts, contribution percentages, and determines net profit, net loss, or break-even. It is designed to give a clear financial summary using basic Python programming concepts.

----- Features -----

Input multiple asset sources, revenue streams, and expenses

Calculates total assets, total revenue, and total expenses

Computes percentage contribution of each source/expense

Determines net profit, net loss, or break-even

Presents a clean financial summary for business insight

----- Technologies Used -----

PyCharm

Fundamental concepts: input(), float(), f-strings, arithmetic operations, round()

----- How to Run -----

Clone the repository

Navigate to the project folder

Run the program

python business_financial_tracker.py

Follow the prompts to enter assets, revenues, and expenses.
